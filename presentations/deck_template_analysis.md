# Deck Template Analysis — `deck_template.pptx`

> Thorough analysis of the 10-slide travel & tour presentation template for reproduction purposes.

---

## 1. COLOR SYSTEM

### Background Colors

| Slide(s) | Background | Usage |
|---|---|---|
| 0, 9 (Cover & Closing) | **Full-bleed photo** with dark overlay | Hero/bookend slides — landscape photography fills the entire slide with text overlaid |
| 1, 2, 3, 4, 6, 8 | **White** `#FFFFFF` | Content slides — clean white background |
| 5 | **White** `#FFFFFF` with a section using **Warm Cream** `#FEF9F1` | Travel Preparation slide — cream-colored accent panel |
| 7 | **White** `#FFFFFF` | Responsible Travel slide |

No slides use a solid dark background on their own — the dark appearance on slides 0 and 9 comes entirely from the full-bleed photography.

### Accent / UI Colors

| Color | Hex | Usage | Frequency |
|---|---|---|---|
| **Black** | `#000000` | Primary text color, shape outlines, most body text | Dominant — used on every slide |
| **White** | `#FFFFFF` | Text on dark/photo backgrounds (slides 0, 9), icon fills, line strokes on cover | Slides 0, 7, 9 |
| **Olive Green** | `#677A1F` | Key accent color — category labels, section markers, "More Info" button border, decorative icon fills, line strokes | Slides 1, 4, 5, 6, 7, 8 |
| **Dark Navy** | `#08172A` / `#09182B` | Freeform decorative shapes on cover/closing slides (near-black with slight blue tint) | Slides 0, 9 only |
| **Warm Cream** | `#FEF9F1` | Background fill for pricing card area on slide 4, accent panel on slide 5 | Slides 4, 5 |

### Color Pattern Summary

The template uses an extremely restrained palette:
- **Primary pair:** Black `#000000` + White `#FFFFFF`
- **Single accent:** Olive Green `#677A1F` — this is the only "brand color" in the template
- **Subtle warm:** Cream `#FEF9F1` — used sparingly as a soft background accent
- **Near-black navy:** `#08172A` — only on cover/closing decorative shapes

**Pattern:** Content slides are white backgrounds with black text. Olive green is used selectively for labels, accent lines, SVG icons, and the "More Info" button. The cover and closing slides use full-bleed photography for visual impact.

### Key Typography Rules

1. **All section titles use Poppins Bold at ~57pt** — consistent across all content slides
2. **UPPERCASE is reserved for feature labels and CTAs** (21pt Poppins Bold) and package tier names (12pt)
3. **Body paragraphs use Roboto 20pt** with 34pt line spacing (1.7x ratio)
4. **The company name sits in the top-center of every slide** at 16.35pt
5. **No letter-spacing adjustments detected** — standard kerning throughout
6. **Line spacing is generous:** titles at ~1.07x, body text at 1.4-1.7x

## Reproduction Strategy

### What to extract from the original file and re-embed:
1. **All 9 SVG icons** — these are custom illustrations that cannot be reliably recreated
2. **All 10 PNG fallback images** paired with SVGs
3. **Freeform organic shapes** from slides 0 and 9 — these have complex custom geometry paths (10 shapes each with intricate bezier curves). Extracting the XML `<a:custGeom>` data and re-inserting is the only reliable approach
4. **All JPEG photos** if maintaining the same imagery

### What can be built from scratch:
1. All text boxes with correct fonts, sizes, colors, and positions
2. Rectangular shapes (photo frames, cream panels, buttons)
3. Line separators (specify weight, color, position)
4. Basic layout grid and spacing
5. Company name bar (recurring element)
6. Grouped elements (icon + label badges) — structure can be recreated, but SVGs inside must be extracted

### What should be simplified or substituted if extraction isn't possible:
1. **Freeform organic shapes** → Replace with simple diagonal or curved rectangles in dark navy
2. **SVG custom icons** → Substitute with standard icon set (e.g., Font Awesome or Material Icons) in matching colors
3. **Photo clusters** → Use simple side-by-side or overlapping rectangular photo frames instead of organic arrangements
