# GoTravel — Brand Style Guide

## Logo Usage

- **Primary logo:** "GoTravel" wordmark with compass icon in teal
- **Icon only:** Compass icon for social media profile pictures and favicons
- **Minimum size:** 120px wide for digital, 30mm for print
- **Clear space:** Always maintain padding equal to the height of the "G" around the logo
- **Never:** stretch, rotate, recolor, or place on busy backgrounds without an overlay

## Color Palette

### Primary Colors
| Color | Hex | Usage |
|---|---|---|
| **Teal** | #0D9488 | Primary brand color — CTAs, headers, links, logo |
| **Warm Sand** | #FBBF24 | Accent — highlights, badges, secondary buttons |
| **Off-White** | #FAFAF9 | Background — pages, email, cards |

### Secondary Colors
| Color | Hex | Usage |
|---|---|---|
| **Deep Navy** | #1E293B | Body text, dark backgrounds |
| **Coral** | #F87171 | Alerts, sale tags, urgency elements |
| **Soft Green** | #6EE7B7 | Success states, eco/sustainability messaging |
| **Light Gray** | #E7E5E4 | Borders, dividers, disabled states |

### Color Rules
- Teal is the dominant brand color — use it for primary CTAs and navigation
- Never use coral for primary buttons (it signals urgency/error)
- Dark backgrounds use Deep Navy, never pure black (#000)
- Maintain WCAG AA contrast ratio (4.5:1 minimum) on all text

## Typography

### Headings
- **Font:** Plus Jakarta Sans (Bold)
- **H1:** 36px / 40px line-height
- **H2:** 28px / 34px line-height
- **H3:** 22px / 28px line-height

### Body Text
- **Font:** Inter (Regular / Medium)
- **Size:** 16px / 26px line-height
- **Max line width:** 680px for readability

### Accent / Labels
- **Font:** Inter (Semi-Bold, Uppercase for labels)
- **Size:** 12-14px with letter-spacing: 0.05em

### Font Rules
- Never use more than 2 fonts in a single piece of content
- Headings are always Plus Jakarta Sans; body is always Inter
- No Comic Sans, Papyrus, or novelty fonts — ever

## Photography Style

- **Authentic over staged:** Real travelers in real moments, not stock-photo smiles
- **Warm, natural lighting:** Golden hour, soft shadows — no harsh flash photography
- **People + place:** Best photos show travelers engaging with a place or culture, not just landscapes
- **Diverse representation:** Show a range of ages, ethnicities, and group types
- **Avoid:** Overly filtered or heavily saturated edits; tourist cliches (feet-by-the-pool, airplane-window shots)

### Photo Treatment
- Apply a slight warm tone (+5 warmth, -10 highlights) for consistency
- Use teal/sand color overlays on hero images when placing text on top
- Round corners (8px radius) on all non-hero images

## Iconography

- Style: Outlined, rounded, 2px stroke
- Color: Teal (#0D9488) on light backgrounds, White on dark backgrounds
- Consistent sizing: 24px default, 32px for feature highlights

## Layout Principles

- **White space is sacred** — let content breathe; don't cram
- **Mobile-first** — design for phone screens first, then scale up
- **Card-based layouts** for trip listings and blog posts
- **Single-column** for long-form content (blog, email)
- **Grid max-width:** 1200px centered

## Social Media Formatting

| Platform | Image Size | Style Notes |
|---|---|---|
| Instagram Feed | 1080x1080 | Photo-forward, minimal text overlay |
| Instagram Stories | 1080x1920 | Bold text, stickers OK, keep key info in safe zone |
| Facebook | 1200x630 | Same as blog thumbnails |
| LinkedIn | 1200x627 | More professional tone, can include data/charts |
| YouTube Thumbnail | 1280x720 | Face + destination + bold title text |

## File Naming Convention

All marketing assets should follow:
```
gotravel-{type}-{topic}-{platform}-{date}.{ext}
```
Examples:
- `gotravel-ad-vietnam-trip-meta-2026-04.png`
- `gotravel-blog-chiang-mai-guide-2026-03.md`
- `gotravel-email-spring-promo-2026-03.html`
